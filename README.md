# Youtube Blog api with Authentication

# Stacks
1. Nodejs
2. Typescript
3. MongoDB
4. Mongoose
5. Typegroose
6. Express
7. Cloudinary


# Run on Development
1. Rename .envExample to .env
2. Goto src/app/constants/index.ts and add your api key and secret to start sending email
3. npm install
4. npm run dev
5. tsc --watch


# Run on Production
1. npm install
2. npm run build
